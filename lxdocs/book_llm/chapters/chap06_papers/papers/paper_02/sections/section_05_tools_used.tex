\section{Explicitly Mentioned Tools/Methods/Algorithms (in paper order)}

\begin{enumerate}
    \item Code Validation Module (CVM) : Checks if code is valid Python using regex
    \item Code Correctness Check Module (CCM) : Traditional test cases + Strict Code Check Chain (SCC)
    \item Code Review Module (CRM) : Main module with subcomponents
    \item Review Necessity Chain (RNC) : Algorithm 1, Decides if review needed (yes/nocorrect/nomeaningless)
    \item Review Generation Chain : Generates actual review comments
    \item LangChain : Manages prompt templates and chains
    \item GPT-4o : The LLM model used
    \item Monaco Editor : Python code editor
    \item Next.js - Web framework (backend/frontend)
\end{enumerate}
