\section{ Common Mistakes }
	\begin{enumerate}
		\item Do not hardcode prompts.
		\item Use environment variables
		\item Using high temperature for factual tasks
		\item Expecting perfect output without prompt structure
		\item Ignoring similarity scores in vector DB search
		\item Using different embedding models for query and storage
		\item Passing retrieval output directly to LLM without inspection
		\item Sending raw chunks without instruction
		\item Using too many chunks
		\item In prompt : Forgetting fallback ("I don't know")
		\item Not printing context during debugging
		\item Assuming retrieval always works
	\end{enumerate}
